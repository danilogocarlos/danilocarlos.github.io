Bee Delivery

A Bee Delivery é uma plataforma do ramo de logística de destaque no mercado brasileiro. Com centenas de milhares de usuários por todo o país, a Bee Delivery disponibiliza aplicativos mobile e web para integrar empresas e entregadores.

Nesse projeto trabalhei como desenvolvedor Back-end e Front-end contemplando as seguintes features:

  • Novo site institucional
    Desenvolvimento integral do o site institucional da Bee Delivery.
  • Novo painel das empresas
    Desenvolvimento das interface de usuário do painel das empresas.
  • Painel do entregador
    Desenvolvimento parcial de interfaces e de funcionalidades do painel do entregador.
  • Painel da franquia
    Desenvolvimento parcial de interfaces e funcionalidades do painel da franquia.
  • Painel de atendimento
    Desenvolvimento das interfaces e funcionalidades do painel de atendimento.

Ferramentas utilizadas: PHP, Laravel, Redis, MySQL, VueJs, Bootstrap 3 e 4, VueJs entre outros.

Bee Mercado

Por causa da pandemia (COVID-19), o comercio inteiro tendo que fechar suas portas, a Bee cria um produto para aumentar as vendas dos pequenos e grande comerciantes locais. O Bee mercado é uma ferramenta que permite o lojista disponibilizar o seu catálogo de produtos online e realize o atendimento via Whatsapp.

No Bee Mercado atuei como desenvolvedor Back-end e Front-end contemplando as seguintes features:

  • Site institucional
    Desenvolvimento parcial do o site institucional do Bee Mercado.
  • Painel da empresa
    Desenvolvimento parcial das interfaces de usuário e funcionalidades do painel das empresas.
  • Plataforma de pedidos
    Desenvolvimento parcial das interfaces e funcionalidades da área de pedidos.

Ferramentas utilizadas: PHP, Laravel, MySQL, VueJs, Bootstrap 4 e VueJs.
